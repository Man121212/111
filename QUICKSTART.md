# 🚀 Быстрый старт - СвайпикЛав

## Этап 1: Подготовка окружения

### Требования
- **Node.js** версии 16 или выше
- **npm** или **yarn** (идёт с Node.js)
- Любой современный браузер

### Проверка установки
```bash
node --version    # должно быть 16+
npm --version     # должно быть 8+
```

## Этап 2: Клонирование репозитория

```bash
git clone https://github.com/Man121212/111.git
cd 111
```

## Этап 3: Установка зависимостей

```bash
cd frontend
npm install
```

Это может занять 1-2 минуты в первый раз.

## Этап 4: Запуск приложения

### Режим разработки (с горячей перезагрузкой)
```bash
npm run dev
```

Консоль выведет:
```
  VITE v8.1.4  ready in XXX ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: http://10.0.0.169:5173/
```

Откройте браузер и перейдите на **http://localhost:5173/**

### Режим продакшена
```bash
npm run build
npm run preview
```

## Этап 5: Использование приложения

### Нижняя навигация (5 табов)

1. **🔥 Открыть** - Главный экран со свайпом профилей
   - Свайпайте влево/вправо
   - Используйте кнопки X, ❤️, 🔥
   
2. **❤️ Лайки** - Профили которые вам лайкнули
   - Первые 4 видны
   - Остальные требуют премиума
   
3. **👥 Мэтчи** - Ваши совпадения
   - Список людей которые вам нравятся
   - Нажмите чтобы открыть чат
   
4. **💬 Чат** - Общение с мэтчами
   - Выберите мэтч из вкладки "Мэтчи"
   - Отправляйте сообщения
   - Используйте эмодзи пикер
   
5. **👤 Профиль** - Ваш профиль
   - Редактируйте профиль
   - Смотрите статистику
   - Выходите из аккаунта

## 🎮 Основные жесты и действия

### На экране Свайпа
| Действие | Результат |
|----------|-----------|
| Свайп вправо | Лайк профилю |
| Свайп влево | Дизлайк профилю |
| Кнопка ❤️ | Лайк |
| Кнопка ✕ | Дизлайк |
| Кнопка 🔥 | Супер лайк |

### В чате
| Действие | Результат |
|----------|-----------|
| Напишите текст | Отправка сообщения |
| Нажмите эмодзи | Добавить эмодзи к сообщению |
| Enter | Отправить |

## 🎨 Темы и кастомизация

```css

  --secondary: #764ba2;    /* Вторичный цвет */

  // ... другие профили

```

## 🐛 Решение проблем

### Ошибка: Port 5173 уже занят
```bash
# Используйте другой порт
npm run dev -- --port 3000
```

### Ошибка: node_modules проблемы
```bash
# Удалите node_modules и переустановите
rm -rf node_modules package-lock.json
npm install
```

### Ошибка: TypeScript ошибки
```bash
# Проверьте типы
npm run type-check

# Если нужно - очистите кэш
rm -rf dist
npm run build
```
