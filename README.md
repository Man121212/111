# 💕 СвайпикЛав - Telegram Mini App для знакомств

Современное и интерактивное приложение для знакомств "СвайпикЛав" созданное как Telegram Mini App. Приложение вдохновлено популярными приложениями для знакомств и предоставляет лучший пользовательский опыт.

## ✨ Особенности

### 🔥 Основные функции
- **Свайп карточек профилей** - Интерактивная система свайпа с анимациями
- **Система мэтчей** - Когда вам нравятся друг другу, это мэтч!
- **Реальный чат** - Общайтесь с вашими мэтчами в реальном времени
- **Просмотр лайков** - Видите, кто вам нравится (с премиумом)
- **Профиль пользователя** - Управляйте своим профилем и настройками

### 🎨 Дизайн и UX
- **Красивый градиент UI** - Современный дизайн с фиолетовыми и розовыми тонами
- **Плавные анимации** - Используется Framer Motion для красивых переходов
- **Адаптивный дизайн** - Отлично работает на всех устройствах
- **Нижняя навигация** - Удобная навигация между разделами
- **Интерактивные кнопки** - Все кнопки имеют hover и tap эффекты

### 🚀 Технологический стек
- **Frontend**: React 18 + TypeScript
- **UI Framework**: Vite + React
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **HTTP Client**: Axios
- **Styling**: CSS3 с поддержкой Grid и Flexbox

## 🛠️ Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Установка

```bash
# Перейти в папку frontend
cd frontend

# Установить зависимости
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173/

### Сборка для продакшена

```bash
npm run build
```

## 🎮 Как использовать

### Табы приложения

1. **🔥 Открыть** - Свайп профилей, лайк/дизлайк
2. **❤️ Лайки** - Посмотреть кто вам лайкнул
3. **👥 Мэтчи** - Список ваших совпадений
4. **💬 Чат** - Общение с мэтчами
5. **👤 Профиль** - Ваш профиль и настройки

### Жесты

- **Свайп вправо** - Лайк профилю
- **Свайп влево** - Дизлайк профилю
- **Нажать кнопку сердца** - Лайк
- **Нажать кнопку крестика** - Дизлайк
- **Нажать кнопку огня** - Супер лайк

## 🎨 Дизайн элементы

### Цветовая схема

## 📱 Адаптивность

Приложение полностью адаптивно:
- **Desktop**: 1200px и выше
- **Tablet**: 768px - 1024px
- **Mobile**: до 480px

## 🔄 State Management (Zustand Store)

### Основные состояния

```typescript
- currentUser: UserProfile | null         // Текущий пользователь
- cards: User[]                           // Карточки для свайпа
- matches: Match[]                        // Мэтчи пользователя
- messages: Message[]                     // Сообщения в чатах
- likedUsers: Set<string>                 // IDs лайкнутых пользователей
- activeTab: string                       // Активный таб
- selectedMatch: string | null            // Выбранный мэтч для чата
- showNewMatch: {show: boolean; user?: User} // Показать новый мэтч
```
